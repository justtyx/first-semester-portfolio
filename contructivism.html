Russian constructivism

Project brief

A mobile view webpage on the topic about Russian Constructivism was made as a first project of the semester. The Gestalt principles, contrast, typography and readability rules were implemented in a webpage design, meanwhile the v was created by using semantically marked up text in HTML and styled with CSS.

Acquired competences

I got acquainted to the first steps of the website creation process, basic design principles that were applied to all upcoming projects of the semester. I learnt to use text editing program Brackets, the importance of marking up code in HTML semantically, basic use of CSS.

Implemented design priniciples

To create the design of the webpage, I applied various design rules such as similarity and proximity laws based on gestalt theory to indicate functions and importance of content. I.e., as images sticking with their captions and headings with paragraphs. I paid attention to typography and readability by choosing sans-serif font ‘Raleway’ and setting font size and line height to keep it comfortable to read.
