The microcomputer

Project brief

The second project of the semester during which a desktop view website about the microcomputer was created. The design was based on the column grid layout system and it was implemented on the website by using CSS grid. The content was marked up with semantically correct HTML code by adding tags indicating purpose of the content.

Acquired competences

I learnt the importance of basing layout of the design on the grid system and implementing it in the CSS. Besides, I learnt more semantic tags in HTML such as header, main, footer and aside. Practised wireframing and using Adobe XD to plan my designs.

Implemented design priniciples

I continued on implementing Gestalt principles, such as proximity and similarity laws in order to keep the unity within the website, i.e. all the headings, paragraphs, images and their captions look the same within their own group, thus helping the user to identify the role of the element. I also kept elements that belong together close to each other (i.e. headings and paragraphs or images and captions).
